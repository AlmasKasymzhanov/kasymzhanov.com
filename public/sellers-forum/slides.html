Часть 1 · делаем вместе сейчас

Разворачиваем за 7 шагов

Программировать не нужно. Вы делаете простые клики и вставляете один готовый текст — остальное делает Claude сам. Идём вместе.

ШАГ 1 / 7 · подготовка

Установить Node.js

Если ещё не установлен — ставим сейчас: на сайте большая кнопка LTS → скачать → далее по умолчанию.

nodejs.org
ШАГ 2 / 7 · подготовка

Установить VS Code

Если ещё не установлен — кнопка Download (сайт сам определит вашу систему), далее по умолчанию.

code.visualstudio.com
ШАГ 3 / 7

Расширение Claude Code

VS Code → слева иконка Extensions (квадратики) → поиск Claude Code → расширение от Anthropic → Install → войти (Sign in).

Важно: именно «Claude Code» от Anthropic — не «Claude Dev».

ШАГ 4 / 7

Скачать архив и распаковать на Рабочий стол

Скачайте kaspi-mcp.zip по ссылке — он попадёт в Загрузки. Перетащите его на Рабочий стол, затем правый клик → «Извлечь всё» (Mac — двойной клик). На Рабочем столе появится папка kaspi-mcp.

ссылка на архив (кликабельна): drive.google.com/file/d/1vCVd3Ehu9vfyF8RTfQMQzRMdxtmH9hta
ШАГ 5 / 7 · ключевой шаг

Открыть папку в VS Code

VS Code → меню File → Open Folder… (Файл → Открыть папку) → выбрать папку kaspi-mcp на Рабочем столе → Open.

Слева появится список файлов проекта. Если спросит «Do you trust the authors?» — нажмите Yes / Trust. Без этого Claude и коннектор не увидят проект.

ШАГ 6 / 7 · один промпт

Открой чат Claude Code и вставь дословно:

Токен: Kaspi Merchant Cabinet → Настройки → API → X-Auth-Token — подставь вместо выделенного.

Это проект kaspi-mcp, папка уже открыта. Разверни его по шагам, объясняя простыми словами: 1) выполни npm install 2) создай .env из .env.example 3) впиши в .env мой Kaspi-токен: ВСТАВЬ_СВОЙ_ТОКЕН 4) запусти npm run sync (данные за 30 дней) 5) скажи, когда коннектор kaspi готов, и дай 4 примера вопросов. Если попросишь разрешение запустить команду — я нажму «разрешить».
ШАГ 7 / 7 · готово

Перезапусти и спрашивай

Claude скажет, что данные загружены. Перезагрузите окно (Ctrl/Cmd+Shift+P → Reload Window), разрешите коннектор kaspi — и спрашивайте обычным языком:

Какая у меня выручка за 30 дней? Где теряю на отменах? Превышен ли порог Kaspi 3%? Топ-10 городов по выручке Сколько идёт через Kaspi Kredit?
Бонус · отчёт для собственника

Готовый HTML-отчёт — одним промптом

Вставь в чат Claude Code:

Сделай отчёт для собственника: выполни npm run report — он создаст файл report.html прямо в папке проекта на Рабочем столе. Затем открой report.html в браузере, чтобы я сразу увидел результат, и в двух словах скажи главные выводы и риски.

Файл report.html — KPI, график выручки, топ-города, структура оплаты, выводы и риски. Откроется сразу, печатается в PDF (Ctrl+P).
